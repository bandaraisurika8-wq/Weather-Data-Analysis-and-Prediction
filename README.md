# Weather-Data-Analysis-and-Prediction

## Dataset
Weather Summary Dataset

## Overview
This project analyzes weather data using Python and Machine Learning techniques. The objective is to explore weather patterns, preprocess data, visualize important insights, and build predictive models.
This was completed as a university group project for theMachine Learning module.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Correlation analysis
- Linear Regression
- Random Forest Regression
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Models
- Linear Regression
- Random Forest Regressor

Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Project Workflow
1. Import dataset
2. Data cleaning
3. Exploratory Data Analysis
4. Data visualization
5. Feature selection
6. Model training
7. Model evaluation
8. Performance comparison

## Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis
- Machine Learning
- Data Visualization
- Python Programming

# Group Project 
This project was completed as part of a university group assignment for the Machine Learning module. We developed two machine learning models to predict weather values and compared their performance using evaluation metrics to identify the model with the highest prediction accuracy.
